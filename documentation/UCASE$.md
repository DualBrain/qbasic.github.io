# UCASE$

Returns a character string with all letters in the specified string expression in uppercase characters.

## Syntax

`UCASE$`(*string_expression*)

## Comments

*string_expression* is any string expression.

## Example

```vb
a$ = "aBcd#F"
PRINT UCASE$(a$)
```

Results in:

```txt
ABCD#F
```
