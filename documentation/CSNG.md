# CSNG

To convert *x* to a single-precision number.

## Syntax

`CSNG(x)`

## Comments

*x* must be a numeric expression (see the [CINT](CINT) and [CDBL](CDBL) functions).

## Example

```vb
10 A#=975.3421222#
20 PRINT A#; CSNG(A#)
```

```text
 975.3421222 975.3421
```
