# STR$

Returns the string representation of a numeric expression.

## Syntax

`STR$`(*numeric_expression*)

## Comments

*numeric_expression* is any numeric expression.

`STR$` is the complementary function to [VAL](VAL).

## Example

```vb
x$ = STR$(3.2718)
PRINT x$
```
