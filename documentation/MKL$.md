# MKL$

Converts a long integer value to a 4-byte string for output to a random-access file by [PUT#](PUT-FILE).

## Syntax

`MKL$`(*numeric_expression*)

## Comments

*numeric_expression* is an long integer numeric expression.

## Example

See [MKD$](MKD$).
