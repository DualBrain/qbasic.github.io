# MKI$

Converts an integer value to a 2-byte string for output to a random-access file by [PUT#](PUT-FILE).

## Syntax

`MKI$`(*numeric_expression*)

## Comments

*numeric_expression* is an integer numeric expression.

## Example

See [MKD$](MKD$).
