# MKS$

Converts a single-precision value to a 4-byte string for output to a random-access file by [PUT#](PUT-FILE).

## Syntax

`MKS$`(*numeric_expression*)

## Comments

*numeric_expression* is a single-precision numeric expression.

See [MKD$](MKD$).
