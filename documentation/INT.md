# INT

Returns the next integer value smaller than or equal to the specified numeric expression.

## Syntax

`INT`(*numeric_expression*)

## Comments

*numeric_expression* is any numeric expression.

## Example

```vb
PRINT INT(99.8), INT(99.1), INT(-99.2)
```

```text
99       99      -100
```
